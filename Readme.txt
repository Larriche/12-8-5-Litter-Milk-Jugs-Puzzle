This is a Java program to print out the solution to the 
12-8-5 milk jugs puzzle described below:

A milkman carries a full 12 liter container of milk.
He needs to deliver exactly 6 liters to a customer
who only has 8-liter and a 5-liter container.

Can he do this? No milk should be wasted.

The solution uses breadth first search as described in Chapter 3 of Artificial Intelligence - A modern approach

The main program is in Solution.java.
